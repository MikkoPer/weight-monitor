# Weight Monitor

Simple PWA targeted on mobile that let's you track your weight on daily basis. No data is send to cloud, entries are stored to browsers indexDB. 

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

For detailed explanation on how things work, check out the [documentation](https://nuxtjs.org).
